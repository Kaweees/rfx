# SO101 Quickstart

If you have a physical SO101 arm, this is the fastest path.

## 1) Setup once

From repo root:

```bash
bash scripts/setup-from-source.sh
```

## 2) Check environment + port

```bash
cli/rfx.sh doctor-so101
```

This checks:

- Python/uv availability
- detected serial ports
- `torch`, `yaml`, `rfx`, and native `rfx._rfx` extension imports

## 2.5) Setup motor IDs + baud (first time)

Run the setup wizard on the direct SO-101 serial bus:

```bash
cli/rfx.sh so101-setup --normalize-ids --set-baud-1m
```

If your local `uv` is unstable, run the script directly:

```bash
.venv/bin/python scripts/so101-setup.py --normalize-ids --set-baud-1m
```

## 3) Run the demo

Auto-detect serial port:

```bash
cli/rfx.sh so101-demo --port auto
```

Explicit port:

```bash
cli/rfx.sh so101-demo --port /dev/ttyACM0
```

This will:

- connect to SO101 through the core robot primitive
- reset to home
- run a gentle sinusoidal motion
- return home on exit

## Bimanual setup (2 arms)

Default bimanual config:

- `rfx/configs/so101_bimanual.yaml`

Run bimanual teleop with one command:

```bash
cli/rfx.sh so101-bimanual
```

Override the config by calling the example directly:

```bash
uv run --python 3.13 rfx/examples/teleop_record.py --config rfx/configs/so101_bimanual.yaml
```

Export options during teleop recording:

- LeRobot:
```bash
uv run --python 3.13 rfx/examples/teleop_record.py \
  --config rfx/configs/so101_bimanual.yaml \
  --export-format lerobot \
  --lerobot-repo-id your/repo
```

- MCAP:
```bash
uv run --python 3.13 rfx/examples/teleop_record.py \
  --config rfx/configs/so101_bimanual.yaml \
  --export-format mcap
```

- Both:
```bash
uv run --python 3.13 rfx/examples/teleop_record.py \
  --config rfx/configs/so101_bimanual.yaml \
  --export-format both \
  --lerobot-repo-id your/repo
```

## Python API (minimal)

```python
from rfx.robot import lerobot

robot = lerobot.so101(port="/dev/ttyACM0")
robot.reset()
obs = robot.observe()
robot.disconnect()
```

## Dataset collection CLI

Collect directly into a LeRobot dataset:

```bash
rfx collect \
  --repo-id your-org/so101-demos \
  --robot-type so101 \
  --episodes 5 \
  --duration 30 \
  --collection-root datasets
```
